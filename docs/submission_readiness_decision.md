# Submission Readiness Decision

## Decision

Workshop-only.

## Rationale

The v2 paper is more honest because it adds a diagnostic probe-noise stress. The result supports the mechanism under clean or lightly noisy probes, but it shows the advantage can disappear when residual signatures are unreliable.

The paper is not ready for a main conference because the evidence is a one-dimensional synthetic simulator with a closed, hand-designed mechanism library and no hardware validation.

## Required Before Main-Conference Submission

- Hardware or high-fidelity contact-rich sim-to-real validation.
- Automatic or learned probe synthesis.
- Robust mechanism classification under realistic sensing noise.
- Out-of-library mechanism detection.
